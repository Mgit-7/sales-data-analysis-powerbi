# 📊 Sales Data Analysis Dashboard (Power BI)

##  Overview

This project analyzes sales data using Power BI to identify trends, performance, and key business insights.

##  Objectives

* Analyze overall sales performance
* Identify top products and regions
* Track trends over time

##  Tools Used

* Power BI Desktop
* Data Visualization

##  Dashboard Features

* Sales trends over time
* Top-performing products
* Regional sales comparison
* KPI indicators
